# Data Analytics Portfolio (R)

## Overview

This repository contains a collection of applied data analysis projects completed using R. The projects focus on statistical modelling, causal inference, data cleaning, and the interpretation of quantitative results in real-world social and economic contexts.

Across all projects, the emphasis is not only on technical implementation, but also on translating statistical findings into meaningful insights relevant for policy and decision-making.

---

## Projects Included

### 1. Education Performance Analysis

A statistical analysis of student performance across subjects using hypothesis testing, data transformation, and segmentation techniques.

**Key methods:**

* Data cleaning and preprocessing
* Summary statistics and variability analysis
* Hypothesis testing (t-tests)
* Group segmentation (performance categories)

**Focus:**
Understanding variation in academic performance and identifying statistically significant differences across subject outcomes.

---

### 2. Does Modernization Impact Democratization?

A causal inference study examining whether economic development (GDP per capita) has a causal effect on democracy levels across countries.

**Key methods:**

* Ordinary Least Squares (OLS) regression
* Mediation analysis
* Instrumental variables (lagged GDP per capita)
* Propensity score matching / covariate balancing
* Robustness checks across multiple models

**Focus:**
Estimating causal relationships between economic development and political regime type while addressing reverse causality, omitted variable bias, and selection bias.
