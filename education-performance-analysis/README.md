# Education Performance Analysis Using R

## Overview

This project investigates patterns in student educational performance using statistical analysis in R. The analysis focuses on subject performance, pass/fail segmentation, hypothesis testing, and practical interpretation of educational outcomes.

## Objectives

* Clean and preprocess educational assessment data
* Analyse subject-level performance trends
* Test statistical significance of score distributions
* Segment students into performance categories
* Translate statistical findings into policy-relevant insights

## Methods Used

* Data cleaning with `complete.cases()`
* Feature engineering and variable construction
* Hypothesis testing using t-tests and proportion tests
* Student performance segmentation using categorical variables
* Normalisation and transformation of scoring systems

## Key Findings

* Reading scores demonstrated greater variability than mathematics and GPS performance
* Students failing mathematics also showed significantly weaker English performance
* Several statistically significant differences had limited practical significance, highlighting the importance of contextual interpretation
* Performance segmentation revealed clustering around pass thresholds rather than extreme failure outcomes
